# Econometric Analysis of Income Disparities among Canadian Immigrants

In this project, we conducted an econometric analysis in R using Ordinary Least Squares (OLS) regression models utilizing data from the 2021 Canadian Census. I included our slide deck for those interested in our results, and included our code to feature how we reached those results! I also included code for the .tex files which are shown in our finalized paper.
